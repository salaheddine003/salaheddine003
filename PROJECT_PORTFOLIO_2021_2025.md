## Standard README Template

Each repository should use this structure:

````markdown
# Project Name

Short one-line project description.

Author: **Salaheddine Abbar**

## Overview

What the project does, why it exists and what skill it demonstrates.

## Technologies

- Python / Java / SQL / Docker / etc.

## Features

- Clear feature 1
- Clear feature 2
- Clear feature 3

## Project Structure

```text
src/
tests/
data/
README.md
LICENSE
```

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
python src/main.py
```

## Example Output

Add screenshots, terminal output or dashboard images when relevant.

## Future Improvements

- Add tests
- Add Docker support
- Add CI/CD

## License

MIT License.
````
